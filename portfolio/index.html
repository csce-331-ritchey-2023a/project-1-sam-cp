While most of my programming work has been on fairly small personal programs, my GitHub has two repositories with some slightly more involved projects.

bigint-cpp is a small library I wrote for incorporating arbitrary-precision arithmetic into C++. In this project, I attempted to leverage fairly simple, yet fast, algorithms for this sort of math, so that C++ can be used to evaluate expressions with numbers larger than 264. One algorithm in particular that I used here was for exponentiation. Rather than simply multiplying a number by itself some number of times according to the exponent, I used a binary approach, where I iteratively squared the base, and multiplied the result by this square if the corresponding bit of the exponent was 1 (similar to the modular exponentiation algorithm, just without the modulus). In the future, I might attempt to implement even faster, more efficient algorithms to make the computations even quicker and easier.

deflate is a C implementation of the DEFLATE compression algorithm commonly used in ZIP archives. While at the moment it can only "inflate" (decompress) data, it would be interesting to add compression capability. The project began when I was reading an ePub file, and noticed that the ePub reader on my Windows machine was not very user-friendly. I wanted to design a better ePub reader for Windows, but to do this, I needed to understand the format. This led me down the rabbit hole of learning about Huffman coding and the DEFLATE algorithm. I originally attempted to implement the algorithm in C++, but it was not going well, so I decided to remake it in C, since I assumed it might be more portable that way. This project has been a great experience in learning to read and implement standards (in this case, RFC 1951), as well as in programming in a non-object-oriented language—even though I still used many object oriented techniques. There is still a long way to go before this can be a full-fledged ePub reader, but decompressing the files is a good start.
